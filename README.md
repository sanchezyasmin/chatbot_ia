# Converse com a Solar: AI Companion com CLI e ADK do Antigravity

Solar é uma AI Companion, uma companheira de IA visual e interativa. Você envia uma mensagem e ela irá responder com texto e voz em mais de 10 idiomas. Além disso, ela é capaz de detectar em qual idioma
você está enviando mensagens para responder de forma correspondente (o idioma da voz, no entanto, deve ser alterado pelo próprio usuário). 

O projeto inicial foi criado a partir do workshop "**Construa seu Primeiro Companheiro de IA: Um Workshop para Iniciantes**" promovido pela **Google Developer Groups** em 2026. Nele, aprendemos a instalar e registrar uma 
habilidade a CLI Antigravity, ativar recursos de programação com o ADK (Agent Development Kit), desenvolver uma personalidade para o agente e adicionar o ```google search``` para que ele pudesse responder sobre eventos recentes.

O esqueleto desse projeto foi criado pela Christina Lin.

## 🛠️ Ferramentas utilizadas
- **Linguagem**: Python
- **Interface visual**: HTML, CSS e Javascript
- **Hospedagem do back-end**: render.com
- **Criação do avatar**: desenhado à mão no Medibang Paint
- **Ferramentas adicionais**: CLI Antigravity, ADK (Agent Development Kit)

## ✂️ Adições ao projeto original
### 🔇 Adicionada opção de mutar o agente com reflexo no avatar
O projeto agora possui opção de mutar a voz do agente, com reflexo no avatar (assim que ativado, apenas a imagem char-mouth-closed é utilizada, enquanto o agente responde apenas por mensagem).
### 🎨 Criação de um avatar exclusivo feito à mão
Para aperfeiçoar o projeto, um avatar exclusivo foi desenhado à mão para a agente Solar. Foi utilizada o sofware gratuito Medibang Paint para a criação das artes, que foram feitas por mim.
### ☁️ Hospedagem do back-end do projeto em nuvem
Como o Github Pages tem funcionalidade apenas visual, foi necessário hospedar o back-end do projeto em uma plataforma de nuvem. O projeto foi simplificado para não ultrapassar o limite gratuito fornecido
e hospedado no site render.com.
### 🔧 Ajustes visuais na interface visual do site
Ajustes visuais no HTML e CSS também foram implementados como melhoria do projeto original.
### 📱 Design responsivo
Agora o projeto conta um design responsivo e 100% funcional na versão mobile. Na falta de vozes da Google disponíveis no navegador mobile, será listada todas as vozes disponíveis no sistema.
